#Face recognizer

tool for recognizing face patterns from webcam stream.

##Installing and running
  - git clone https://github.com/Xtinct/face_recognizer
  - make
  - ./face_recognizer

To remove executables, use **make clean** command

##Dependencies
  - libsdl(debian-like) or sdl(arch)
  - v4l-utils
